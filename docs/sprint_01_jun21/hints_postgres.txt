- Iniciando postgres WINDOWS

- Logando usuario postgres:

sudo -u dev psql.

npm run watch // para iniciar execução do ts
npm run dev // para startar o servidor


No MacOs: psql pa_dev para entrar no banco pa_dev e rodar as consultas desejadas

\dt para listar todas as tabelas

Para limpar o terminal do postgres:

Mac/ Linux:

\! clear 

Windows

\! cls


Criando um backup do banco localmente:

pg_dump -h localhost -p 5432 -U postgres pa_dev > backup.sql

Restore:

pg_restore -U postgres -d pa_dev -c backup.sql
