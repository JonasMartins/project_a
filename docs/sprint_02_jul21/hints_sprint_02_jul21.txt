

1) Criando from com next-js usando o tamplate chakra-ui


yarn create next-app --example with-chakra-ui app_name

yarn add -D typescript @types/node



2) URQL ( Back )

Urql premite usar hooks com as mutations 
dentro dos componentes react


Installing urql

yanr add urql graphql
yarn add -D @types/urql


3) graphql code generator, ( Front )

yarn add -D @graphql-codegen/cli

initialize wizard:

yarn graphql-codegen init 
e responder o questionário para geração dos arquivos


4) Add typescript-urql plugin para o codegenerator


yarn add -D @graphql-codegen/typescript-urql


5) Após todo esse setup, criar uma pasta graphql no src
do front, que na configuração inicial do congen

"src/graphql/**/*.graphql"

- Todo arquivo.graphql aqui ao usar 

yarn gen (gen é o nome do comando setado do codegen)

- será gerato todos os arquivos, hooks e types corretos

na pasta generated
